# Python×COVID-19
このページでは、Pythonを用いて鳥取県の新型コロナウイルス感染者数を毎日午前10時（JST)に自動で取得しその推移をグラフを作成し公開しています。実行環境の都合につき更新が遅れる場合があります。

ここにある内容について、いかなる責任も負えません。ご利用になられる際は、ご了承ください。
